Point at The Magic Cat !!!
---

This is an offline backup copy of the Magic Cat Academy game by Google.
Specially, it is designed to play with hand rather than play with mouse.
Before playing, ensure that the device has available camera and stay at bright places.

Demo
=========
This demo was shown on SITCON (Students’ Information Technology Conference) in 2026 in Taipei.

We recommend to play this demo on PC or notebook rather than portable devices.

Playing the game offline
=========
To get started, you will need to host your own HTTP server.

Run at `http://localhost:4000/magic-cat-academy-Google/combine.html`
